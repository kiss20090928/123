     元素名稱  { 屬性1 : 值1;
                 屬性2 : 值2;
                 ............}

     如:  H1  { color : red ; 
                text-align : center }
  
     元素名稱1,元素名稱2,..  { 屬性1 : 值1;
                               屬性2 : 值2;
                               ............}

     如:  H1,H2  {color : red ; 
                 font-style : italic;
                 text-align : center }
其中元素名稱稱為 選擇器
{屬性 : 值 }稱為 規則
